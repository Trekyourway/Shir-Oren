# Shir Oren Site Structure

זוהי שלד סטטי עבור אתר Shir Oren, לפי המבנה שביקשת.

## מבנה תיקיות

- `index.html` – דף הבית הראשי של האתר הכולל קישורים לעולמות השונים (קואוצ'ינג, AI, מסעות).
- `about/` – דף אודות (`index.html`).
- `contact/` – דף צור קשר (`index.html`).
- `ai/` – עולם הבינה המלאכותית עם תתי-מדורים:
  - `articles/` – מאמרים.
  - `learn/` – חומרי לימוד.
  - `tools/` – כלים ושירותים.
- `coaching/` – עולם הקואוצ'ינג:
  - `index.html` – דף הבית של העולם (כאן מומלץ להעתיק את תוכן ה־HTML המלא שיש לך).
  - `products/` – דף מוצרים כללי (`index.html`) ותתי־מוצרים:
    - `co-work/` – מוצר Co-work.
    - `90-minutes-out-of-fog/` – מוצר 90 דקות לצאת מהערפל.
  - `articles/` – מאמרים בנושא קואוצ'ינג.
  - `learn/` – חומרי לימוד בנושא קואוצ'ינג.
  - `self-check/` – בדיקה עצמית:
    - `burnout/` – שאלון Burnout.
- `treks/` – עולם המסעות:
  - `index.html` – דף הבית של עולם המסעות.
  - `articles/` – מאמרים בנושא מסעות.
  - `learn/` – חומרי לימוד בנושא מסעות.
  - `solo-hiking/` – דף טיולים סולו.
- `assets/` – תיקיה לרכיבים משותפים כמו תמונות, אייקונים, פונטים, CSS ו־JS.

כל קובץ `index.html` הוא עמוד ראשי של התיקיה שלו. התוכן בדפים הוא בסיסי וניתן להחליף אותו בהתאם לצורך.
